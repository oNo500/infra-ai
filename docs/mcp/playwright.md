# Playwright MCP

基于 Playwright 的浏览器自动化。让 Claude 无需编写测试文件即可导航页面、
填写表单、点击元素、截图。

**激活范围：项目级** —— 仅用于需要浏览器交互的项目。

## 何时激活

- 由 Claude 驱动浏览器的 E2E 测试工作流
- 抓取或视觉验证类任务
- 功能实现后的 Web 应用验证

## 何时不要激活

- 纯 API 项目 —— 会增加约 12k token 的无用工具描述开销
- 一次性浏览器任务改用 `agent-browser` CLI（零 MCP 开销）

## 上下文开销

约 12k token。只在会话确实要用到浏览器时才激活。
