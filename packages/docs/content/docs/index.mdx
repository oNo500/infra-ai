## 🎯 这是什么？

本项目提供了一套完整的 AI 辅助编码基础设施：

- **Skills**: 可复用的性能优化和最佳实践指南
- **Agents**: 预配置的 AI 代理，包含合并后的完整规则集

## 📚 关于 Skills

Skills 是针对特定技术栈的性能优化和最佳实践指南。每个 skill 包含：

- 详细的规则说明
- 正确与错误的代码示例对比
- 影响分析和优先级
- 实际应用场景

## 🤖 关于 Agents

Agents 是预配置的 AI 代理文档，将相关 skills 的所有规则合并为单一文档，便于：

- AI 工具直接使用
- 快速查阅完整规则集
- 团队协作和知识共享
