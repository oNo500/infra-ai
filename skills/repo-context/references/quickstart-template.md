## AI 填写指南

- **从 package.json 脚本中取命令** — 使用实际脚本名，不要猜测
- **只写确认存在的命令** — 仅收录文档中有记录或明确存在的命令
- **按使用频率排序** — 最常用的工作流放在最前面
- **非显而易见的步骤说明原因** — 例如为什么要在启动前先跑数据库迁移
- **删除不适用的章节** — 无数据库则删除数据库初始化；无环境变量则删除该章节
- **前置条件以实际要求为准** — 检查 `.nvmrc`、`engines` 字段或 README 中的版本要求
