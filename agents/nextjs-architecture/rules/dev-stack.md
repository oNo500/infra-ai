## 技术栈

核心框架:
- React 19 + Next.js 15 + TypeScript 5.9

UI 组件:
- 优先级：shadcn/ui -> Tailwind CSS
- 图标库：Lucide React
- 基础：Radix UI

样式方案:
- Tailwind CSS 4

状态管理:
- 服务器状态：React Query
- 客户端状态：Zustand

工具链:
- 包管理器：pnpm
- API Mock：MSW (Mock Service Worker)
- 代码规范：ESLint (扁平化配置) + TypeScript 严格模式
